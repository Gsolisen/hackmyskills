The quiz session needs the question content (prompt, answer, command) to render cards, but the Card ORM model only stores scheduling fields. The question content lives in the YAML bank (loaded by `load_questions()`).

**Architectural decision (Claude's discretion):** Load ALL question dicts once at session start inside `run_session()` and build a lookup dict `questions_by_id: dict[str, dict]` keyed on question id. Pass this dict to each handler. This avoids per-card YAML re-reads.

Update `run_session()` signature to build this lookup:
```python
from hms.loader import load_questions
# inside run_session():
all_questions = load_questions()
questions_by_id = {q["id"]: q for q in all_questions}
```

Then pass `questions_by_id[card.question_id]` to each handler call (add `q_data: dict` as second parameter to all handler functions, shifting `session` to third position):
`_handle_flashcard(card, q_data, session, _readkey)`

**_handle_flashcard(card, q_data, session, _readkey=None):**
```
_render_progress(console, card_num, total)    # called by run_session before dispatch — not inside handler
console.print(f"[dim][{card.topic} · {card.tier}][/dim]")
console.print(Panel(q_data["prompt"], border_style="blue", title="[dim]Question[/dim]"))
console.print("[dim]Press any key to reveal answer...[/dim]")
_wait_for_key(_readkey=_readkey)  # any keypress flips
console.print(Panel(q_data["answer"], border_style="green", title="[dim]Answer[/dim]"))
console.print("[dim]Rate your recall:  1=Again  2=Hard  3=Good  4=Easy[/dim]")
key = _wait_for_key(valid_keys={"1","2","3","4"}, _readkey=_readkey)
rating_map = {"1": Rating.Again, "2": Rating.Hard, "3": Rating.Good, "4": Rating.Easy}
rating = rating_map[key]
persist_rating(card, rating)
session.record(card.topic, rating.value)
```

**_handle_command_fill(card, q_data, session, _readkey=None):**
```
console.print(f"[dim][{card.topic} · {card.tier}][/dim]")
console.print(Panel(q_data["prompt"], border_style="blue", title="[dim]Fill in the command[/dim]"))
user_answer = input("$ ").strip()
canonical = q_data["command"].strip()
correct = user_answer.lower() == canonical.lower()  # case-insensitive exact — no fuzzy (user decision)
if correct:
    console.print("[bold green]✓ Correct![/bold green]")
    rating = Rating.Good
else:
    console.print(f"[bold red]✗ Incorrect.[/bold red]  Correct answer: [cyan]{canonical}[/cyan]")
    rating = Rating.Again
persist_rating(card, rating)
session.record(card.topic, rating.value)
```

Note: `_render_progress()` is called by `run_session()` before dispatching to handlers, so handlers do NOT call it themselves. The `_readkey` parameter is passed through to `_wait_for_key()` for test injection.

Also update all four handler stubs to accept `(card, q_data, session, _readkey=None)` signature.
